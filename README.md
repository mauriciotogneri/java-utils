# Java Utils
A collection of Java utility classes.

## Installation

Add the following code to your **pom.xml**:

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

and the dependency:

```xml
<dependency>
    <groupId>com.github.mauriciotogneri</groupId>
    <artifactId>java-utils</artifactId>
    <version>0.2.0</version>
</dependency>
```
